## 🛰️ Featured Project: Mars Cargo Optimization
A diagnostic system designed to validate launch parameters for heavy-lift rockets. This project serves as the foundation for my understanding of procedural logic and method-driven development in Java.

### 🧠 Key Technical Concepts Applied
* **Method Definition & Invocation**: Implementing modular logic within the class structure and executing it through the `main` entry point.
* **Functional Versatility**: Utilizing method arguments to allow customized data input (Weight, Fuel, Systems status) at runtime, enhancing code reusability.
* **Logic Encapsulation**: Moving complex control flow sequences into dedicated method bodies to improve readability and maintainability.
* **Data Flow Management**: Leveraging `return` statements to communicate system states and specific error codes back to the caller.
* **Comprehensive Diagnostics**: Implementing sequential logic to evaluate multiple failure points simultaneously, providing a detailed feedback report for engineering teams.

* **Standard**: Clean Code & Domain-Driven Logic
